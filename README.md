# Manine AI

AI-powered predictive maintenance platform for industrial machines.

## Features

- Machine failure prediction
- Explainable AI using SHAP
- Industrial monitoring dashboard
- Streamlit UI
- Decision Tree ML model
- RandomForest explainability engine

## Tech Stack

- Python
- Streamlit
- scikit-learn
- SHAP
- Plotly
